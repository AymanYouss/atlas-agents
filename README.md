# Atlas

Self-hosted, MCP-native multi-agent platform with a planner-executor-critic architecture.

Atlas takes a goal, plans it, executes it with standards-based tools, verifies the result, and returns a citation-backed report. Runs are durably checkpointed so a crashed run resumes from the last committed step.

> Full documentation, architecture, benchmarks and quickstart follow in this README as the project is built out.
